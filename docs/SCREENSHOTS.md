# Screenshots für die Doku

Diese Dateien binden die READMEs ein (sobald vorhanden). Sprache je Ordner beachten:
deutsche Screenshots nach `docs/de/img/`, englische nach `docs/en/img/`.

Aufnahme über die laufende Instanz (Board im Browser). Für Englisch `&lang=en` anhängen.

| Datei | Ansicht | Aufnahme-URL (Beispiel Test-Board) |
|---|---|---|
| `board.png` | Board-Übersicht | `http://172.30.0.40:8096/?board=team-board` (+`&lang=en`) |
| `card-editor.png` | Karten-Editor (eine Karte anklicken) | dito, dann Karte öffnen |
| `settings.png` | Einstellungen (⚙ → Board/Spalten/Labels) | dito, dann ⚙ |
| `share.png` | „Ansicht teilen" (🔗) | dito, dann 🔗 |

Empfohlen: Fensterbreite ~1200–1450 px, heller **oder** dunkler Modus konsistent.
